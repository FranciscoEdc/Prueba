Para agrupar sistemas operativos FLOSS que considero útil, he decidido crear esta página web utilizando sólo HTML y CSS para que un navegador moderno lo pueda abrir incluso sin JavaScript.
A continuación expongo diversas razones para usar tal software.

1. Transparencia

Puedes ver el código del programa para ver si contiene algo malicioso. Puedes incluso compilarlo si desconfías de los ejecutables que el autor distribuye.

2. Libertad

Tienes la libertad de hacer lo que quieras con ello. Si encuentras un fallo o quieres agregar una funcionalidad, puedes hacerlo, incluso como una derivación del programa.

3. Igualdad

Cualquiera puede acceder al código fuente, incluso si no lo entienden. Cualquier entidad puede auditar el código para comprobar la seguridad de éste.
