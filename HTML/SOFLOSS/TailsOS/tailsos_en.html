TailsOS

Translations:[ES],[PT-EU]

It´s a Debian-based Linux distribution that offers anonimity via Tor, without leaving any fingerprint on the utilised device

1. Routing through the Tor network

To avoid malicious third-parties from spying our internet activity, TailsOS routes everything through the Tor network, besides the own Tor browser.

2. Amnesic OS

TailsOS is installed on an USB drive, from which you boot the OS. Once you shutdown or reboot the PC, all your activity done there will be cleared.

3. Portability

Since you boot the US from an external drive, you can use it on any allowed system.

4. Decent app kit

By default TailsOS contains many apps for any activity you may perform, like the Tor browser, Thunderbird, Pigeon, Monero, etc. If you want to use apps that aren't included, install them on persistent memory.

5. Persistent memory

If you want to save files or apps that aren't included by default, you can create a persistent memory on the external drive to store anything under encryption to preserve it even after shutting-down the PC.
